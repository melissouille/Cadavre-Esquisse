POUR LES PLANCHES :
	Dans le cas où [numéros présent sur doc "maquette/planche_algo_cases.png"]
	(1)small
	(2)square
	(3)horizontal
	(4)medium
	(5)vertical 
	taille A4 = 210x297mm ou pour 300 dpi 2480x3508 pixels

	Définitions variables :
	1page = height = 297mm et/ou 3508 px ET width = 210mm et/ou 2480 px
	1page = min 2ligne (dans le cas d'une présence de au moins 1vertical) ET max 3ligne
	1ligne = 
		3small
		ou 1small + 1medium
		ou 2square
		ou 1horizontal
		ou 1vertical + 1square (??? enchainement en colonne  ???)
	SI ligne actuelle = 1ligne
		ALORS ligne == pleine
	
	SI ligne == pleine
		ALORS nouvelle ligne
	SI 1page == 3ligne pleines (? voir cas avec (5) ?)
		ALORS page == pleine
	SI page == pleine
		ALORS 
			TANT QUE numéro page actuelle < nombre max de page
				FAIRE nouvelle page
				SINON bd terminée (à voir possibilité d'ajouter des pages)

Cases stockées en tableau pour naviguer plus facilement il sera créé lors de la création de la BD 
l [ jusqu'a 3 cases]
p [ jusqu'à 3 l]

	l = numéro ligne 
	l >= 1 ET <= 3
	p = numéro page
	p >= 1 ET <= nb_page (nb_page saisit lors de la création de la bd, peut être modifiable par le créateur)
	utilisation d'une boucle à travers un tableau à plusieurs dimensions
		?? FOR ou FOREACH ?? {
			SI ligne[l] = finie {
				ajout ligne[l+1]
			}
			SI case[0] == null ou false {
				ALORS utilisateur peut choisir small + square + horizontal + medium + vertical
				enregistrement ligne[l] (incrémentation de l) de page[p] (incrémentation de p)
			}
			SINON SI case[c] == small {
				case[c] = small ET case[c+1] = small ou medium 
				SI case[c+1] == small {
					ALORS case[c+2] == small
				}
				SINON case[c+1] == medium ALORS ligne[l] = finie
			}
			SINON SI square {
				case[c] == square ET case[c+1] == square ET ligne[l] = finie
			}
			SINON SI horizontal {
				case[c] = horizontal ET ligne[l] = finie
			}
			SINON SI vertical {
				case[c] = vertical ET case[c+1] = square ET case[c+1].ligne[l+1] = square
				(? ou 2square alignés verticalement ?)
			}

RESERVATION

	alors affiche modal reservation "Case réservée ..."
	sinon affiche bouton "Réserver cette case"

SI case[c] en cours
	alors affiche bouton "Réserver cette case"

Si case[c] réservé par utilisateur identifié
	 alors affiche bouton "Annuler réservation et libérer la case"
